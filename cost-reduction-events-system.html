Implementation Strategy

Visual Hierarchy

  • Discount Badge: Eye-catching position (top-right)
  • Strikethrough Price: Shows original cost clearly
  • Green Discount Price: Highlights the savings
  • Savings Tag: Quick mental math for players
  • Animated Elements: Draw attention without being annoying

Discount Stacking Rules

  1. Base event discount (e.g., 50% Energy Rush)
  2. + Category bonus (e.g., 10% Science buff)
  3. + VIP/Premium bonus (e.g., 10% subscriber)
  4. + Special promo (e.g., 15% new player)
  5. = Maximum 90% discount cap

Psychological Triggers

Urgency

  • Countdown timers
  • Flash sale animations
  • "Limited time" messaging

Value

  • Show savings amount
  • Highlight percentage off
  • Compare to regular price
