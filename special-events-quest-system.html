Implementation Guidelines

Special Events

  • Corner ribbons for visibility
  • Animated borders for urgency
  • Countdown timers when applicable
  • Enhanced reward displays

Streak System

  • Persistent badge with flame icon
  • Shows current streak + multiplier
  • Glows/pulses at milestones
  • Integrates with reward calculations

Quest Progress

  • Bottom progress bar (non-intrusive)
  • Hover for detailed progress
  • Special badge for quest cards
  • Completion animations
