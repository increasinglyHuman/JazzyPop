Recommended Approach

  1. Default to "Show All" - Let players see everything available, with clear visual distinctions
  2. Use Progressive Disclosure - Simple at first glance, detailed on hover/tap
  3. Consistent Visual Language - Green = affordable, Yellow = close, Red = unaffordable
  4. Actionable Feedback - Don't just say "can't afford", suggest how to get resources
  5. Respect Player Choice - Some want to see everything, others want a cleaner view
  6. Animation & Feedback - Gentle pulse on affordable cards, shake on failed attempts
