## Common Emoticon Shortcuts

| Emoji | Name | Common Shortcut |
|-------|------|----------------|
| 😀 | Grinning Face | `:D` or `:grin:` |
| 😃 | Grinning Face with Big Eyes | `:smiley:` |
| 😄 | Grinning Face with Smiling Eyes | `:smile:` |
| 😊 | Smiling Face with Smiling Eyes | `:blush:` or `:)` |
| 😂 | Face with Tears of Joy | `:joy:` |
| 😍 | Smiling Face with Heart-Eyes | `:heart_eyes:` |
| 😎 | Smiling Face with Sunglasses | `:sunglasses:` or `B)` |
| 😭 | Loudly Crying Face | `:sob:` |
| 😡 | Pouting Face | `:rage:` |
| 🤔 | Thinking Face | `:thinking:` |
| 🙄 | Face with Rolling Eyes | `:roll_eyes:` |
| 😴 | Sleeping Face | `:sleeping:` |
| 🤮 | Face Vomiting | `:vomiting:` |
| 🤯 | Exploding Head | `:exploding_head:` |
| 🥳 | Partying Face | `:partying_face:` |

---

*Note: Names shown are the official Unicode CLDR (Common Locale Data Repository) short names. Some platforms may use slightly different names or shortcuts.*
