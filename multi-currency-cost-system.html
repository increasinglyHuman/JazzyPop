Implementation Considerations

Visual Clarity

  • Color coding for affordability
  • Obvious OR separators
  • Requirements vs costs distinction

Smart Defaults

  • Energy as primary currency
  • Show best payment option first
  • Hide OR if player can't afford alternative
  • Auto-select cheapest available option

  • Show what player has on hover
  • Clear "can't afford" messaging
  • Suggest how to get more resources
